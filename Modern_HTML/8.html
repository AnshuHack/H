What is HTML?

HTML stands for Hypertext Markup Language. It is a standard markup language for creating web pages. HTML is used to tell web browsers how to structure a web page's content and words.

What is Semantic Tags?

Give meaning to the content of a web page. Making it Easier for Search Engines, Screen Readers and others tools to Understand the Page and its content.

List of the semantic tags in HTML along with their description :-

1.Header : It specifies header for a document or any section.

2. Footer: It specifies foot of the document or any section.

3. Navigation: It defines the navigation link and it is used to navigate in document via links.

4. Main : It defines the main content.

5. article: It defines the part of the main content with paragraph and heading.

6. Section : It also defines the part of the main content with paragraph and heading.
