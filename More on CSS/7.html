Google Fonts

Google Fonts is a service that offers free and open-source fonts for use on the web and in computers. Google Fonts includes an interactive web directory for browsing the library, as well as APIs for using the fonts via CSS and Android. Some of the best Google Fonts include: Roboto, Open Sans, Lato, Montserrat, Oswald. Some other free fonts available on Google Fonts include: DM Sans, Inter, Space Mono, Space Grotesk, Work Sans, Syne, Libre Franklin, Cormorant. Google Fonts is a computer font and web font service owned by Google. This includes free and open source font families, an interactive web directory for browsing the library, and APIs for using the fonts via CSS and Android. Google Fonts is also used With the Google Workspace.
