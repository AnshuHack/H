List of tags

  • h1: It define the heading level 1
  • p: This tag is used to write a paragraph
  • div: It creates a container, so that you can merge other tags within
  • b: It makes the text bold
  • em: It is used for emphasis the text
  • s: It is used for strike through
  • u: It is used for underline the text
  • nav: It is used for navigation
  • table: It is used for creating table
  • a: anchor tag is used for linking web pages.
